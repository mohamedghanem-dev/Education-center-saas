# سنتر الدروس CRM

نظام إدارة سنتر الدروس المتكامل — Electron.js Desktop App

## الميزات
- ✅ إدارة المجموعات (اسم، مادة، مدرس، أيام، وقت، رسوم)
- ✅ إدارة الطلاب (بيانات كاملة، رقم ولي الأمر، صف، مجموعة)
- ✅ تسجيل الحضور والغياب + أرشيف الحصص
- ✅ المدفوعات الشهرية (مدفوع / جزئي / لم يدفع)
- ✅ الاختبارات والدرجات مع تقدير تلقائي (A/B/C/D)
- ✅ ملف شخصي لكل طالب (حضور، مدفوعات، درجات)
- ✅ زر واتساب لكل طالب (ولي الأمر + الطالب)
- ✅ QR code لكل مجموعة
- ✅ تقارير: حضور، مدفوعات، درجات
- ✅ تفعيل ECDSA P-256 مرتبط بـ Hardware ID
- ✅ وضع فاتح + داكن
- ✅ نسخ احتياطي واستعادة

## توليد مفتاح ترخيص
```bash
pip install cryptography
python3 generate_license.py --hwid "HARDWARE_ID" --days 365
```

## بناء التطبيق
```bash
npm install
npm run build:win
```
